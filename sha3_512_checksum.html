SHA3-512 File Hash

Calculate a SHA3-512 hash from a local file or file URL. Local files are processed in your browser. Read more

What is SHA3-512?

SHA3-512 is the fixed 512-bit SHA-3 hash standardized in NIST FIPS 202. This calculator processes raw file bytes. SHA3-512 is separate from SHA-512 and from the original Keccak-512 File Hash.

How to use this file hash calculator

Choose a local file or enter a URL and select the output format. Local files remain in your browser. A remote file can be read only if its server permits cross-origin requests.

Result example

A file containing exactly abc produces b751850b1a57168a5693cd924b6b096e08f621827444f70d884f5d0240d2712e10e116e9192af3c91a7ec57647e3934057340b4cf408d5a56592f8274eec53f0 in Hex. Filename and metadata are excluded.

Important limitations

Matching a SHA3-512 digest detects changed bytes but does not authenticate the publisher. Obtain the expected value independently through a trusted channel.

Frequently asked questions

Why does my result differ from another tool?

Confirm that both tools use SHA3-512, not SHA-512 or Keccak-512, and process identical raw bytes.

Can I use the result to verify a download?

Yes, when the expected SHA3-512 digest comes from a trusted publisher. For encoded text, use the SHA3-512 calculator.
